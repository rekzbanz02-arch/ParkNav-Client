ParkNav — User Guide
Xavier University · Ateneo de Cagayan
🅿️ What is ParkNav
📡
Real-Time Parking Monitor
ParkNav shows live parking availability across all XU campus lots. Data is pulled automatically from physical sensors every few seconds — no manual refresh needed.
🔝 Header Indicators
🟢
System Online / Connection Lost
The pulsing dot shows system status. Green = live data flowing normally. Red = sensor connection lost; displayed numbers may be outdated.
⏱️
Countdown Timer (e.g. "3s")
Counts down to the next automatic data refresh. It resets to 3 seconds after each successful fetch. Orange color means a fetch is currently in progress.
🕐
Live Clock
Shows the current local time, updated every second. Use this to cross-check the "last updated" timestamps on each card.
🚧 Campus Entry — Main Gate
🚗
Vehicles on Campus
Total vehicles counted through the XU main gate that have not yet exited. This is a campus-wide perimeter count, not a lot count.
🚦
Traffic Badge
Indicates current gate congestion level:
● Normal — below 80% of baseline   ● Busy — 80–99%   ● High Traffic — 100%+
ℹ️
Why Gate Count ≠ Lot Totals
Vehicles doing drop-off only, or cars still moving within campus, appear in the gate count but are not yet assigned to a parking lot. This is normal and expected.
📊 Parking Lots — Combined Overview
🔢
Large Number = Available Slots
The big figure on the left is the total number of free parking slots across all monitored lots right now.
🍩
Donut Chart
Visually shows how full campus parking is overall. The ring turns green when space is available, amber when filling up, and red when near or at capacity.
📋
Occupied / Total Capacity / Lots Full
Quick at-a-glance breakdown. Lots Full turns red when one or more parking areas have zero available slots.
🗂️ Individual Lot Cards
🏛️
Campion Hall · Museo de Oro · Loyola House
One card per monitored parking area. Each shows available slots, total occupied vs. capacity, a fill bar, and a status badge.
📶
Color Bar at Top of Card
A quick visual cue: green = slots free, amber = filling up (≥85% full), red = completely full (100%).
🏷️
Status Badges
✓ Available — slots open   ! Filling Up — ≥85% occupied   ✕ Full — no slots left
🕓
Last Updated Timestamp
Shows exactly when the sensor last sent data for that specific lot. If this time is several minutes old, the sensor for that area may be temporarily offline.
💡 Tips for Finding Parking
1️⃣
Check the Combined Overview first to see if any slots are available before heading in.
2️⃣
Use the individual lot cards to identify which specific area has the most open slots.
3️⃣
If the Gate shows High Traffic but lots show availability, there may be vehicles still in transit — expect slight delays.
4️⃣
Data refreshes automatically every ~3 seconds. There is no need to reload the page.
